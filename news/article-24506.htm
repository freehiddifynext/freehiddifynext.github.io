FailureAnalyzer可以检查异常中特定的异常，这样有助于我们处理异常

import org.springframework.boot.diagnostics.AbstractFailureAnalyzer; import org.springframework.boot.diagnostics.FailureAnalysis;  public class IllegalArgumentFailureAnalyzer extends AbstractFailureAnalyzer<IllegalArgumentException> {      @Override     protected FailureAnalysis analyze(Throwable rootFailure, IllegalArgumentException cause) {         return new FailureAnalysis(cause.getMessage(), "Please check startup argument!", cause);     } }

其次，在 src/main/resources 路径下新建 META-INF/spring.factories，在 spring.factories 文件中添加一下内容

org.springframework.boot.diagnostics.FailureAnalyzer=com.study.cxk.studyFailureAnalyzer.IllegalArgumentFailureAnalyzer com.study.cxk.studyFailureAnalyzer表示包名
